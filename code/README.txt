Java WEb第一次作业：
班级信息展示中心
功能：当用户登录成功后，页面有相关学生的姓名、照片等,点击图片时能够转到那个人的简介页面
（登录之后才能够查看，如果在未登录的情况下通过粘贴网页地址打开 的话则跳转到登录页面）
（电子稿，打包成压缩文件 ,命名规则如：计算机1302姓名31306020**.rar）

Java Web第二次作业：
在第一次作业的基础上加入Hibernate框架，静态页面改成动态页面，页面的学生资料数据、用户信息通过
数据库进行存取，页面上加入注册用户功能、根据条件查询学生功能，查询到的结果动态展示。

项目使用说明：
1.安装mysql，在mysql里建立一个名为hibernate的数据库
2.更改src路径下的配置文件hibernate.cfg.xml中
<property name="connection.password">54hizengbiao</property>
的54hizengbiao为自己电脑上mysql数据库的密码并保存
3.运行src/test下的Test.java文件
4.运行整个项目
5.输入用户名、密码，点击注册按钮
6.输入用户名、密码，点击登录
